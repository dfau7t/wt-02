About the Project

The Generative Music and Visuals project is an exploration of the intersection between music, art, and technology. Using innovative algorithms and programming techniques, this project generates unique musical compositions in real-time while creating captivating visualizations.

Through this project, I aim to push the boundaries of what is possible in both the auditory and visual realms. By leveraging the power of code, I create dynamic and immersive experiences that engage the senses and evoke emotions.
